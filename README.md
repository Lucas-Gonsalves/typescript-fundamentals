# TypeScript Fundamentals

This repository is a collection of TypeScript examples and exercises for learning the fundamentals of TypeScript. It's designed as a training resource to understand basic concepts like primitives, types, interfaces, generics, and utility types.

## Structure

- **primitives/**: Examples of basic TypeScript types such as arrays, functions, objects, unions, and more.
- **personalized/**: Custom type definitions including interfaces, enums, generics, intersection types, and type assertions.
- **utilities/**: Utility types like `Partial`, `Pick`, `Omit`, `Record`, `keyof`, and `typeof`.
